Contributing to pdfout's documentation

The documentation is maintained as Pod in the doc/ subdirectory.

Building the documentation is performed by make.pl.

The TOC

The table of contents of the documentation is maintained in doc/index.pod.

Links

For internal links we use the following mechanism:

 L<Description|file-name>

, where file-name does not have a '.html' or '.pod' suffix.

To link to a subheading, use

 L<Description|file-name#name of subheading>

Linking to a subheading in the current file is done with

 L<Description|#name of subheading>
